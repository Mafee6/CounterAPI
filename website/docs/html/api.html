CounterAPI Endpoints

/api Root Endpoint for CounterAPI
/create?name Create a counter without the create page
/counter/:id Get a counter by its ID
/counter/:id/add Increment a counter by its ID
/counters/name/:name Get all counters by their names
/counters/count Get the total number of counters registered.
/widget/:id/:color(?format=json) Get the counter widget that will no increment on request. (Next Topic has all the colors)
/widget/:id/:color/add(?format=json) Get the counter widget and increment it.
/view/:id View the counter in a Graphically Represented Form.
