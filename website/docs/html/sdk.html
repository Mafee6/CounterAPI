Widgets - Using Our SDK

We have learnt how to import the SDK, now we are going to learn how to implement it.

Making the counter Div:

  1. Make an empty Div

  2. Set the Div's Class to "counter"

  3. Add an Attribute named counter with the value as your counter ID

  4. Add an attribute named counter-color with the value as a valid color
  5. Add an attribute named counter-add with the value as an empty string
  6. Done!
