Counter Data

Content Type: JSON

Here are the Properties of the response:

name Counter Name
id The ID of the counter
url The URL to fetch the counter
increment The URL to increment the counter thro. the API
widget The Widget URL of the counter
widgetAdd The Widget URL of the counter that also increments the counter (handy)
randomColor A random Widget Color assigned to the counter
timestamp The timestamp when the counter was created
count The current count of the counter.
lastIncrement The timestamp when the counter was last incremented
schemever The Schema Version of the counter
You can view your counter by:
/api/counter/:counter-id/

Example: /api/counter/j9nb7q28kg/

Current Schema Version Is: 1.26
